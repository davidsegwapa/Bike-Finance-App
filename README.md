Bike Finance App 

Overview

The Bike Finance App is a frontend web application designed to manage bike financing and tracking logic through a simple user interface. It simulates bike payment progress, ownership tracking, and finance management using client-side logic only.

The application runs entirely in the browser using HTML, CSS, and JavaScript. There is no backend server or database connected.


Key Features
	•	 Add and manage bikes in the system
	•	 Track weekly or monthly payments
	•	 Monitor finance progress per bike
	•	 Assign users or drivers to bikes
	•	 Simulated ownership transfer after completion of payment period
	•	 View payment history records
	•	 Dashboard interface for managing bikes and data


Tech Stack
	•	Frontend: HTML, CSS, JavaScript
	•	Storage: Browser-based data handling (JavaScript memory / localStorage)
	•	Backend: None
	•	Database: None


How It Works
	1.	The user opens the application in a browser.
	2.	Bikes are added through the user interface.
	3.	Payment data is entered and managed using JavaScript logic.
	4.	The system calculates and updates finance progress in real time.
	5.	Ownership status changes automatically once payment conditions are met based on predefined rules.


Installation & Setup

1. Clone the repository

git clone https://github.com/davidsegwapa/bike-finance-app.git
cd bike-finance-app

2. Run the application

Open the application directly in a browser:

open frontend/index.html

Or double-click the index.html file.


Important Notes ⚠️
	•	The application runs entirely in the browser.
	•	Data resets when the page is refreshed unless stored in localStorage.
	•	There is no backend or external database.
